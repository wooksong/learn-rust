# My Rustlings Progress

Learning Rust with [Rustlings](https://github.com/rust-lang/rustlings)

## Setup

```bash
# Install rustlings CLI (v6.5.0)
cargo install rustlings@6.5.0

# Run
cd Rustlings
rustlings
```
